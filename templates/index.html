Criptografia de Áudio e Imagem

Sistema web desenvolvido em Python para codificar, distorcer, enviar e recuperar arquivos de áudio e imagem.

Sobre o projeto

Este projeto tem como objetivo aplicar conceitos de oscilações, ondas, processamento digital de sinais e criptografia em arquivos de áudio e imagem.

O sistema permite que o usuário envie um áudio ou imagem, escolha uma senha de codificação e gere um novo arquivo distorcido. Para recuperar o áudio original, é necessário utilizar a mesma senha usada no processo de codificação.

Entrada

O usuário envia um arquivo de áudio no formato WAV ou imagem.

Codificação

O arquivo é distorcido e combinado com uma chave pseudoaleatória.

Envio

O arquivo codificado pode ser baixado ou enviado por e-mail.

Recuperação

O arquivo codificado é restaurado usando a mesma senha da codificação.

Arquivos Aceitos

Os arquivos de áudio são aceitos somente no formato .WAV, e os arquivos de imagem são aceitos nos formatos .png, .jpg, .jpeg, .bmp

Senha de App

Se pretender enviar o arquivo via Gmail, use uma senha de app criada para isso, diferente da senha normal do e-mail.
